# 📚 Documentation

This directory contains community health and governance files for the **CyberSkill** organization.

## Contents

| File | Description |
| ---- | ----------- |
| [CONTRIBUTING.md](CONTRIBUTING.md) | Guidelines for contributing to CyberSkill projects |
| [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) | Contributor Covenant Code of Conduct (v2.1) |
| [SECURITY.md](SECURITY.md) | Security vulnerability reporting and disclosure policy |

## How These Files Work

These files are placed in the **organization-level `.github` repository**, which means they serve as **defaults for all repositories** under `cyberskill-world`. Individual repos can override them by placing their own versions in their respective `.github/` or `docs/` directories.

> For more details, see [GitHub's documentation on default community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).
